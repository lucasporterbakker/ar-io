Protocols

We build the tools that help you build the products.

The ar.io team is developing an expanding set of protocols poised for widespread adoption on the permaweb.

  • Build apps on top of ArDrive with ArFS - Arweave File System
  • Give permaweb data a friendly name with ArNS - Arweave Name System
  • Use the open-source io Network Protocol and help run the backbone of the permaweb through a decentralized network of gateway operators

    What do you want to build?

    The ar.io team is behind a growing number of perma products. Our flagship data storage product, ArDrive, accounts for more than 10% of the weave with increasing user adoption. ArDrive is now available as a CLI, web app, and a mobile app.

    Other products are in the works such as Turbo (a fiat payment service), Arweave Name Tokens (ANT), and ar.io Gateways.

    Community members are already working on their own tuned ar.io Gateways such as Permagate.

    Permaweb

    ar.io is the front door to the permaweb.

    What if you never lost a file again? What if the web belonged to you and not big tech? Arweave’s permaweb is a globally distributed network that puts the power back in your hands.

    Step inside to explore what you can do with permanent data and its unmatched level of incentives and customization.

    The IO Token

    The token that powers the permaweb.

    Used for governance, profit sharing, staking and payment services like Arweave Name System (ArNS). It will incentivize ar.io gateways to operate under a low-trust model with the users of their service.
