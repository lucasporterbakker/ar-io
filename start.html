Are you a Gateway Operator?

The Arweave protocol doesn‚Äôt exist on its own.

The various nodes on the AR.IO Network, known as Gateways, are the interface between users and the Arweave network. Each Gateway acts like a ‚ÄúPermaweb Service Provider‚Äù and supports multiple, value-added, low-trust services for users and applications that need to read and write data on Arweave. Gateway services are modular, extendable, and configurable to include:

  • seeding layer one transactions to the greater Arweave network
  • caching and serving chain and transaction data
  • resolving friendly names to transaction data with the Arweave Name System
  • indexing and querying layer one and two transactions
  • proxy for Arweave nodes serving the Arweave HTTP API

Gateway operators keep the front doors to the permaweb open by running reliable and performant systems for their own use cases, their community or even the whole Permaweb.

Get in on the action. Becoming a gateway operator is currently one of the biggest opportunities in the Arweave ecosystem. A good gateway operator will be able to generate revenue as they increase network traffic, optimize their servers, and take part in network governance. and follow us on Twitter to track the release of the ArIO gateway protocol.

Are you a Developer?

Arweave is easier, cheaper and more fun to build on than most developers ever imagined.

Within the Arweave community there are many great development tools to help build your app such as arweave.js, warp contracts, and the Community Labs library of tools.

One of the premier tool for building on Arweave is the Arweave File System (ArFS). ArFS enhances the Arweave protocol by adding the ability to create data hierarchy (folders), private encryption, file versioning, controlled sharing and more. It lets you harness the power of the permaweb for the projects you have in mind.

Are you an io Token Holder?

The IO Token powers the permaweb. IO token holders play a crucial role in making permanent data available to everyone.

For example, token holders will be able to stake their tokens with AR.IO gateways, helping support a robust gateway ecosystem and giving them a chance to take part in governance and generate revenue.

IO Token Holders will also be able to lease ArNS names so they can buy and trade permaweb URLs.

Last but not least, token holders will have a say in the overall governance of the ar.io network and will help shape our dynamic community. The ar.io network will become what we make of it together!
